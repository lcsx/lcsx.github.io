# 校验参数

使用koa-parameter 422错误校验结果

安装cnpm i koa-parameter --save 使用

...
const parameter = require('koa-parameter');
...
//放在请求体后面
app.use(bodyparser());
app.use(parameter(app));
routing(app)
...

控制器中使用参数校验

class lcs{
	...
	index(ctx){
		ctx.verifyParams({
			name:{type:'string',required:true/*默认true*/},
			age:{type:'number',required:false}
		})
	}
}

如果不满足要求，会返回422状态码，以及具体报错
